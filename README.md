# 🗻 富士山实时天气 · 登山参考

一个面向登山者的富士山实时天气网页应用：多高度实时天气、登顶安全评估、推荐登顶窗口与御来光（日出）观赏概率。无需安装、无需注册，打开网页即用，可添加到手机主屏当 App 用。

**在线访问：** https://yxmanyxman.github.io/Mt.Fuji-Weather/

> ⚠️ 本应用为登山参考工具，天气数据为模型预报，仅供参考。出发前请务必同时核对各登山道官网与日本气象厅的官方信息，并量力而行、注意安全。

## 功能

- **多高度实时天气**：同屏显示山脚（河口湖一带）、所选登山道五合目、山顶（剑峰 3776 m）的气温/体感、风速、阵风、风向、能见度、降雨量、云量。
- **登顶安全评估**：以山顶为准，综合阵风、雷暴/降水、体感温度、能见度，给出「✅ 条件良好 / ⚠️ 需谨慎 / ⛔ 不建议登顶」三档判断与具体原因。
- **推荐登顶窗口**：扫描未来 7 天逐小时数据，自动挑出风小、无降水、能见度好的连续时段，并标注是否可看御来光。
- **御来光观赏概率**：依据山顶日出时段的云量与降水估算日出观赏概率（极佳 / 较好 / 一般 / 较低 / 渺茫）。
- **四条登山道切换**：吉田口、富士宫口、须走口、御殿场口，各按其起点海拔与朝向取数据，重点突出能见度与降雨量。
- **24 小时趋势图 + 未来 7 天预报**。
- **PWA**：可「添加到主屏」像 App 一样使用；Service Worker 缓存外壳，弱网/离线可打开。
- **数据兜底**：网络异常时显示上次成功缓存的数据，并标注最后更新时间。
- 每 10 分钟自动刷新，也可手动刷新。

## 四条登山道

| 登山道 | 起点海拔 | 朝向 | 特点 |
| --- | --- | --- | --- |
| 吉田口 | 约 2305 m | 北麓（山梨县） | 设施最完善、山小屋与救护点最多，最热门，初次登山首选 |
| 富士宫口 | 约 2380 m | 南麓（静冈县） | 起点最高、距顶最近但坡度大，上下同路 |
| 须走口 | 约 1970 m | 东麓（静冈县） | 低段森林遮蔽好、较清静，本八合目与吉田道汇合 |
| 御殿场口 | 约 1440 m | 东南麓（静冈县） | 起点最低、距离最长最难，山小屋稀少，适合体力强者 |

## 部署（GitHub Pages）

1. 将本仓库所有文件上传到 GitHub。
2. 进入仓库 **Settings → Pages**，Source 选择 **Deploy from a branch**，分支选 `main`、目录选 `/ (root)`，保存。
3. 等 1–2 分钟，访问 `https://<用户名>.github.io/<仓库名>/` 即可。

> PWA 的「安装到主屏」与离线缓存需通过 https 网址访问才生效（GitHub Pages 自带 https）；本地双击打开文件不会出现安装按钮。iPhone 用 Safari 打开后，通过「分享 → 添加到主屏幕」安装。

## 数据来源

天气数据来自 [Open-Meteo](https://open-meteo.com/)（免费、无需 API key，非商业用途）。图表使用 [Chart.js](https://www.chartjs.org/)。

## 风速安全参考

| 风速 | 评级 |
| --- | --- |
| < 8 m/s | 良好 |
| 8–12 m/s | 注意（阵风影响平衡、易失温） |
| 12–18 m/s | 危险（强烈不建议登顶） |
| > 18 m/s | 极度危险（约 8 级以上，禁止登顶） |
